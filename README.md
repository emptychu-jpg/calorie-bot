# 🍎 Калорій Трекер Бот

Telegram бот для відстеження харчування з аналізом фото їжі через Claude AI.

## Можливості

- 📸 Аналіз фото їжі (калорії, БЖУ, клітковина)
- 📊 Статистика за день/тиждень/місяць
- 💡 Поради щодо користі/шкоди продуктів
- 🇺🇦 Повністю українською мовою

## Швидкий старт

### 1. Встановлення Python

Переконайся, що у тебе встановлений Python 3.10+

```bash
python --version
```

Якщо немає — завантаж з https://python.org

### 2. Клонування та налаштування

```bash
# Перейди в папку з ботом
cd calorie-bot

# Встанови залежності
pip install -r requirements.txt
```

### 3. Налаштування ключів

Створи файл `.env` (або відредагуй код напряму):

```bash
# Скопіюй шаблон
cp .env.example .env

# Відредагуй .env та встав свої ключі
```

Або просто відкрий `bot.py` та заміни:
- `ТВІЙ_TELEGRAM_TOKEN` на токен від @BotFather
- `ТВІЙ_ANTHROPIC_API_KEY` на ключ від Anthropic

### 4. Запуск

```bash
python bot.py
```

Готово! Тепер відкрий свого бота в Telegram і надішли фото їжі 🎉

## Команди бота

| Команда | Опис |
|---------|------|
| `/start` | Почати роботу |
| `/today` або `/сьогодні` | Статистика за сьогодні |
| `/week` або `/тиждень` | Статистика за 7 днів |
| `/month` або `/місяць` | Статистика за 30 днів |
| `/help` | Допомога |

## Деплой на Railway (безкоштовно)

1. Зареєструйся на [railway.app](https://railway.app)
2. Натисни "New Project" → "Deploy from GitHub"
3. Завантаж код на GitHub і підключи репозиторій
4. Додай змінні середовища:
   - `TELEGRAM_TOKEN`
   - `ANTHROPIC_API_KEY`
5. Railway автоматично задеплоїть бота
